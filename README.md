## 📜 Description
mnemoria is a smart solution for persisting memory storage tailored for AI agents. With features like hybrid semantic and full-text search, it offers an efficient way for AI applications to store and retrieve data. The append-only binary format ensures durability, while zero-copy reads enhance performance, making it an excellent choice for anyone looking to manage AI memory effectively.
